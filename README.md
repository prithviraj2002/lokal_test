# lokal_test

## Project Overview
1) model directory contains model class for news being fetched from the news api.
2) provider directory contains providers news provider and saved provider.
3) screens directory contains the screens used in app.
4) widgets directory contains reusable widgets used in app.
5) The app begins from main.dart file.

## Features implemented
1) To display headlines from 6 different categories of news.
2) To get headlines from different countries.
3) To search news.

## Additional features implemented
1) To save an article.
2) To unsave an article.
3) Splash screen.

### Packages used
provider: ^6.0.5
http: ^0.13.5
animated_splash_screen: ^1.3.0
dio: ^5.1.2
